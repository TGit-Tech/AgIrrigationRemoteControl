1.INTRODUCTION


The “Ag Irrigation Pump Remote Control” project's goal is to provide RF remote control and monitoring features of an Agriculture-Industry Irrigation Pump; typically used in wheel-line and hand-line irrigation setups. The project uses Xbee RF radio's with up to a 28-mile range ( Specification of Xbee; but more practically when using at ground level - a mile or two ). The project consists of a Hand-Held “Hand Remote” and a pole mounted “Pump Controller” as well as high voltage control electrical components to be mounted inside the pumps electrical panel.

B.LCD Keypad Shield


  1. Solder two 10K Resistors

    1. One from VIN to A1

    2. One from A1 to GND


The two resistors setup a voltage divider from the 9Vdc battery (which we will attach to VIN). Analog inputs on the Arduino should never exceed 5Vdc. The voltage divider will split the battery voltage to 1/2 its actual voltage. This will allow the hand-remote to monitor the battery level.


  1. Solder Voltage Wires from ICSP Pin-Out

    1. One (Red) wire from ICSP 5V Pin

    2. One (Black) wire from ICSP GND Pin


These wires will feed the 5Vdc to the 'Active Buzzer' and the 'Xbee Shield' by its ICSP port. The Xbee Shield also implements a reset button; but we will not be attaching it since its usage shouldn't be needed.





C.Zigbee/Xbee Shield


  1. Solder Xbee Shield to the LCD Keypad ( Flipped upside down )

    1. Solder all pins into the holes of the LCD Shield


  1. Proceed to step 'D' – Install Active Buzzer then proceed below


  1. Connect Power to Xbee Shield

    1. Cut to length, Strip and Plug (Red) wire into the ICSP Plug of the Xbee Shield as shown in picture above

    2. Cut to length, Strip and Plug (Black) wire into the ICSP Plug of the Xbee Shield as shown in picture above


  1. Now Install the LCD, Buzzer, Zigbee Combo into the Front Case


  1. Remove Green Xbee Shield Jumpers and Solder their Center Pins to LCD Keypad ( or Arduino UNO as pictured ) D8 & D9 as shown below



  • Resources

    • https://www.arduino.cc/en/Main/ArduinoXbeeShield

    • https://www.arduino.cc/en/Guide/ArduinoXbeeShield


  • Jumper Details

    • Xbee Position = Xbee UART Serial connects to Arduino Serial

    • USB Position = Xbee UART Serial connects to USB Serial ( Computer )

    • The Jumpers are aligned Top->Bottom Exactly the same as the Xbee DIN/DOUT are aligned on the shield (DIN is closest jumper to Digital Pins Jack on Shield)


Since the Arduino uses Serial-0 for uploads and debug communication; we use the 'Software Serial' library and Digital Pins D8 & D9 for Serial Communications to and from the Xbee module.


D.Active Buzzer


  1. Solder Buzzer 'S' pin to LCD Keypad Shield D2

  2. Solder (Red) wire to Center pin

  3. Solder (Black) wire to the Left Pin


The 'S' pin is bent back to connect to the solder joint on the LCD Shield. The Red/Black wires were not cut but the insulation was split using automatic wire strippers. The Center and Left Pins of the Buzzer were cut just long enough to solder wires to them.

4.2.2Customizing


  • Overview

    • Downloaded firmware includes menu items for;

      • Pump Controller Selection - Ability to control multiple Pump Controllers from one Hand-Remote Unit

      • Power - Sets or Read the Pump Power as either ON or OFF

      • Water Level - An Accessory IO-Monitor to check the canal/ditch water level

      • Battery - Reads the Battery Voltage Level

      • Pressure - An Accessory IO-Monitor to check the water pressure

      • Alarms - Various Alarms for the monitoring readings above


The menu items are defined in the Setup-Menu Function identified by line 'void SetupMenu() {'. Each item in the Menu has a numeric index (i.e. Menu[index-#]).


  • To delete a Menu Item

    • Find the menu items index (ie. Menu[index]) you'd like to delete by identifying it by its' Menu[Index].Text setting

    • Select ALL Menu[index] lines with the same index

    • Press delete

    • Starting from the top make sure the indexes are in order; for example if you deleted [3], change [4] to [3], [5] to [4] and etc.. for all entries

    • Up towards the top of the program ( ~line 50 ) change value of '#define MENUITEMS ?' to the number of menu items that exist now


  • To add a Menu Item

    • Start a new line below the last one defined and enter the attributes that need to be set

      • Menu[?].Text - The text that will display on the LCD

      • Menu[?].Value - Typically not set; holds the value of the item

      • Menu[?].ValueSettable - TRUE/FALSE determines if the user can change the value

      • Menu[?].ValueLocation - PROG/LOCALPIN/REMOTEPIN/EPROM; Determines where the value of the item is set/retrieved

      • Menu[?].Pin - Arduino Pin# associated with the item

      • Menu[?].Analog - TRUE/FALSE; Default is FALSE, Set to 'true' if using an Analog pin for the value

      • Menu[?].AlarmCompare - LESS/GREATER/EQUAL; Defines how an alarm will compare its limit value to the ValueIdx Value

      • Menu[?].AlarmValueIdx - The Menu[Index].Value this alarm item will compare itself with

      • Menu[?].Option[?].Poll - TRUE/FALSE; Default is true. Set to false if the value doesn't need to be refreshed and displayed


Options are how a limited amount of select-able items are defined for a value. For example ON or OFF would be two options.


      • Menu[?].Option[?].LastOption - Counting from 0-> ; How many options are defined

      • Menu[?].Option[?].Text - Option[?].Text defines an option that can be selected like 'ON'

      • Menu[?].Option[?].DrivePin - Option[?].DrivePin LOW or HIGH when this option gets selected


    • Up towards the top of the program ( ~line 50 ) change value of '#define MENUITEMS ?' to the number of menu items that exist now

5.3.2Install Power Supply


BE SURE THE MAIN BREAKER IS TURNED OFF AND CAREFULLY CHECK THAT THERE IS NO VOLTAGE ON THE BREAKER WITH A PROPER METER!!!

480VAC is deadly; and can cause severe flash burn (e.g. A meter on the wrong setting can “flash” explode causing “flash” burns on you even at a distance away)

See https://www.youtube.com/watch?v=6hpE5LYj-CY


  1. Cut DIN Rail to length that will comfortably fit the Power Supply and Solid State Relay

  2. Using tapped screws fasten the DIN rail horizontally in an open space within the Pump Panel

  3. Attache the RHINO power supply to the DIN rail

  4. Wire L1, L2, L3 of the Power Supply to the Main Breaker (Bottom disconnected side)


5.3.3Install Solid State Relay


  1. Attach the Solid State Relay next to the Power Supply on the DIN Rail

  2. Wire Connector 15 → Auto Position of HAND-OFF-AUTO switch

  3. Wire Connector 16 → M auxiliary contact on the right side of the starter



5.3.4Install Pump Controller


  1. Mount the 'Pump Controller' Box up high on the telephone pole or on-top of the Pump Panel


  1. Wire the Solid State Relay (A1) Connector to the Pump Controller D5 Pin

  2. Wire the Solid State Relay (A2) Connector to the GND on the Pump Controller

  3. Wire the 24Vdc(+) from the Power Supply to the Pump Controller

  4. Wire the 24Vdc(-) from the Power Supply to the Pump Controller
