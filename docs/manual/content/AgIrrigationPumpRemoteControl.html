2.1.3Arduino Xbee Shield

  • Resources

    • This Schematic is the right one https://www.arduino.cc/en/uploads/Main/XbeeShieldSchematic.pdf

    • https://www.cooking-hacks.com/documentation/tutorials/xbee-arduino-raspberry-pi-tutorial/

    • http://www.cooking-hacks.com/skin/frontend/default/cooking/pdf/arduino_XBee.pdf



    • (1) XBee Pro S3B


  • Zigbee Shield ( Two Jumpers labeled Xbee / USB )


    • Xbee Position ( Xbee UART ↔ Arduino UART )

      • Xbee Data Out (DOUT) → RX (D0) Arduino Uno

      • Xbee Data In → TX (D1) Arduino Uno

Note: In the Xbee Position; the Xbee UART Serial communications are connected to the Arduino board as well as

the Atmel processors UART which is also still connected. This results in the Xbee UART Serial data being shown on

the Serial Monitor Utility but it also prevents sketch uploads from the Computer to the Arduino.

Therefore these jumpers must be removed before uploading an Arduino sketch onto the Arduino.


    • USB position

      • Xbee Data Out (DOUT) → RX (D0)

      • Xbee Data In (DIN) → TX of the FTDI

      • The Xbee module communicates directly with the computer ( NOT REALLY, the Arduino micro-controller has to be removed )

    • NOTE

      • Since Arduino uses Serial-0 for uploads and communication with the computer the smartest thing to do is use Software Serial and two other Digital pins and connect them to the Xbee Shield instead of trying put both the Computer and Xbee on the same Serial device

      • DIN Xbee is the closest jumper to the PIN JACK.

3.Pump Station


  • Features

    • Back-up Arduino power source for when power is lost ( Track time off and turn-back-on automatically )

      • Back-up battery level indicator and possible charge circuit

    • Water Level ( Ultrasonic or Toilet Float?)

    • Timer and Logger of events

    • Water Pressure Transducer

    • Water Flow Meter (If affordable)


  • Automation

    • Automatically turn off the pump if no water exists

    • Automatically re-start pump if a power failure has occurred

    • Set to Automatically turn on pump when water height is okay


3.1Primary Relays ( Relay Shield v1.3 )


  • Relay Activation Pins = D4, D5, D6, D7

  • Xbee Interface = Xbee I/O Pins DIO0-DIO3 are also connected to the 4-Relays

  • In order to Connect Tx/Rx of the Xbee to the Arduino a Wire Jumper must be soldered into placed

    • DOUT (Xbee-2) → Rx (Arduino-D0)

    • DIN (Xbee-3) → Tx (Arduino-D1)
